1) What is DOM and what is the purpose of DOM?

Ans: The full form of DOM is "Document Object Model".It is how a browser shows a web page internally.


2) How will you select HTML elements using DOM? Name the DOM methods.

Ans:There are five ways to select html elements using DOM.We have to use selector..
1.getElementById()
2.getElementsByTagName()
3.getElementsByClassName()
4.querySelector()
5.querySelectorAll()


3) What is event bubble?

Ans:when the event handlers are invoked and one element is nested on to the other element and are part of the same event.
This technique or method is called Event Bubbling.


4) What is a callback function and why will you use it?

Ans: we pass a function inside another function and call that inside the parent funcion that is called a "Callback function".
When we want one function to execute only after another function has completed executing , in that case we can use a "Callback function".
